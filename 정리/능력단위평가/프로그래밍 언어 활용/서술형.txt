오류 : 
for문 안에서 index가 0부터 5까지 돌아가는데 numbers 배열의 인덱스는 4까지만 있으므로 인덱스 범위가 넘어가는 오류가 발생
해결 방안 :
for문 조건식에서 index <= 5 를 index < 5 로 바꾸면 해결된다

2.
오류 :
int calculationResult = numerator / denominator;
이 부분에서 denominator는 0인데 나눗셈 연산에서 0으로는 나눌 수가 없기 때문에 오류가 발생한다.

System.out.println("0으로 나눌 수 없습니다.")
뒤에 세미콜론을 찍지 않아 오류 발생

해결 방안 :
denominator를 0이 아닌 다른 정수로 바꾸면 해결된다.
출력문 뒤에 세미콜론을 찍어서 해결

3.
오류 : 
for문 안에 있는 console.log(10 / numbers[index]); 부분에서
for문은 index가 0~4까지 반복하는데 맨 처음 index가 0일 때 console.log(10 / 0); 이므로 0으로 나눌 수 없기 때문에 오류 발생

해결 방안 :
for문 안에 index 변수를 1부터 시작하거나 numbers[0]의 값을 0이 아닌 다른 정수로 바꾸어서 해결한다.
